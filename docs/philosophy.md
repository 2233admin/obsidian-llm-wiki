# vault-mind 设计哲学

> 知识不是被管理的，是自己运动的。

## 政治经济学基座 (马克思)

### 生产力与生产关系

vault-mind 的根本矛盾：生产力 (adapter + compiler + agent) 与生产关系 (vault 的组织结构) 之间的矛盾。

当生产力发展到 vault 结构容不下时 -- 不是加功能能解决的，是要改 vault 的组织方式。

| 概念 | 映射 |
|------|------|
| 生产资料 | adapter 体系 (MCP, filesystem, obsidian) |
| 劳动过程 | compiler (感性 -> 理性的飞跃) |
| 生产关系 | vault 组织结构 + concept graph topology |
| 上层建筑 | skills + agent + dreamtime |

### 异化与去异化

笔记写下的那一刻就开始异化 -- 脱离写作者的意图和语境，变成"死文件"。

obsidian-second-brain 的 skill 不解决异化，只是更高效地操作异化物。

vault-mind 的 compile = **去异化 (Aufhebung der Entfremdung)**：把散落笔记重新接回认识-实践循环，让"死知识"重新成为"活的认识"。

### 剩余价值

每次编译必须产出比输入更多的知识 (surplus)。compile 只是换格式存储 = 没有价值生产 = 假劳动。

surplus = 新的 relationships + 新的 contradictions + 新的 synthesis。

### 再生产

系统不只生产知识，还必须再生产知识生产的条件本身：

- compile 产出 concept -> concept 指导下次 emerge 方向 -> emerge 发现新 pattern -> 新 pattern 成为下次 compile 的 schema
- 这是**扩大再生产**，不是简单重复

## 矛盾论基座 (毛泽东)

事物发展的根本原因在事物内部的矛盾性。vault 的自运动不靠外力（cron/用户调用），靠内在矛盾。

| 概念 | 映射 |
|------|------|
| 主要矛盾 | evaluate() 找当前 vault 最阻碍知识发展的矛盾 |
| 次要矛盾 | 排队等主要矛盾解决后升级 |
| 矛盾转化 | 主要矛盾解决 -> 次要矛盾升级为新主要矛盾 -> 持续运动 |
| 对抗性矛盾 | 两个 concept 直接冲突 -> 必须调和，不能共存 |
| 非对抗性矛盾 | 视角不同但不冲突 -> 综合为更高层次 |
| 矛盾的特殊性 | 每个 vault 的主要矛盾不同 |

## 实践论基座 (毛泽东)

通过实践发现真理，通过实践证实和发展真理。

```
感性认识 (raw notes)
    -> 理性认识 (compiled concepts)
    -> 实践 (用知识做决策/写代码/交易)
    -> 结果写回 vault
    -> 新的感性认识
    -> cycle
```

实践是检验真理的唯一标准：concept 指导的决策失败 -> 写回 vault -> 触发重编译 -> 理性认识被修正。

## 群众路线基座 (毛泽东)

从群众中来，到群众中去。

```
散落笔记 (群众)
    -> compile + emerge (集中)
    -> feed back to daily notes, skills, decisions (到群众中去)
    -> 日常使用产生新笔记 (再从群众中来)
```

## 核心设计原则

1. **矛盾驱动，不是清单驱动** -- agent 不遍历优先级列表，分析主要矛盾
2. **编译是飞跃，不是整理** -- 从感性到理性的质变，不是换个格式存
3. **编译必须产出剩余** -- relationships + contradictions + synthesis，否则是假劳动
4. **实践检验，不是自洽即真** -- concept 必须被实践验证，失败的必须修正
5. **去异化是核心使命** -- 把死笔记接回认识-实践循环
6. **群众路线，不是精英设计** -- 散落笔记是知识的源泉，不是待清理的垃圾
7. **内在运动，不是外力维护** -- vault 的自组织靠矛盾推动，不靠用户勤奋
8. **扩大再生产** -- 每次编译同时再生产下次编译的条件

## evaluate() 三论合一

```
矛盾论: 知识体系内部有什么对抗?
    -> 对抗性矛盾 (必须调和)
    -> 矛盾转化 (什么升级了)

实践论: 认识-实践循环在哪里断裂?
    -> concept 从未被实践引用 = 死知识
    -> 决策失败但 concept 未修正 = 理性落后于实践
    -> 大量 raw 未编译 = 感性堆积，飞跃被阻断

群众路线: 集中和回到群众的循环通不通?
    -> 散落提及 3+ 次没 concept = 群众声音被忽视
    -> concept 从未出现在 daily notes = 集中了没回到群众
```

三论综合判断 -> 决定行动。
