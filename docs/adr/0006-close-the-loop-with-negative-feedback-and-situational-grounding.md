# Close the work-driver loop with negative feedback; promote knowledge claims by situational grounding rather than blanket human review

Amends invariant §0 #3 and ADR 0005's "knowledge claims always go to human review." The Work Driver is a closed-loop negative-feedback controller (after Tsien's *Engineering Cybernetics*), not a generator→evaluator→human classifier: an independent skeptic evaluator measures each Work Run against a reference (C2 = a `done-check`, C3 = a grounding-confidence signal), and the controller actuates one of auto-promote / feed-back-and-regenerate / escalate-to-human according to the error. Human review is the high-error or non-convergence escalation path, not the default sink — a system that escalates everything is an open loop with a human as actuator.

Knowledge claims (C3) are **not** permanently downgraded as machine-authored. Instead, when the situation warrants — a claim will be reused as grounding, its grounding signal is ambiguous, it is high-stakes or conflicts existing truth, or it is stale — the controller spends an external cross-verification action (calling `opencli` to search real data and query genuinely heterogeneous models) to reduce hallucination, then re-measures; a claim that clears situational cross-verification earns equal standing. This guards against error-compounding (auto-promoted truth fed back as grounding and amplifying hallucination, per the gemma-challenge multi-agent observation) by re-verifying at the point of reuse rather than by static distrust. Consensus must use decorrelated models plus external anchors, since high consensus among correlated models can be collective hallucination.

Rollout follows gain-tuning: auto-promotion starts off (everything escalates) and its threshold is lowered only as the observable escalation-rate and consistency-drift sensors stay low. The trade-off accepted: a bounded chance of a machine-authored claim entering current-truth — held down by conservative observable thresholds, caught by drift sensors, and reversible by git — in exchange for a loop that actually closes instead of dumping every result on a human.
