# Simple Test

This file contains various link types for testing.

## Wikilinks

- Simple: [[Target Note]]
- With alias: [[Target Note|display text]]
- With heading: [[Target Note#Heading]]
- With block: [[Target Note#^block-id]]

## Markdown Links

- Simple: [Target Note](Target%20Note.md)
- With alias: [display](Target%20Note.md)
- URL encoded: [Target](Target%20Note.md)

## Embeds

- Basic embed: ![[Target Note]]
- With heading: ![[Target Note#Heading]]
