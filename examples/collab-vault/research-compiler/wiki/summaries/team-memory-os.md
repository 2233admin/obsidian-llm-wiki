# Summary: team-memory-os

> Source: `raw/team-memory-os.md`
> Compiled: 2026-05-16

## Overview

The source defines LLMwiki as a [[team-memory-os]] loop: raw material is captured cheaply, compiled into an inspectable wiki, queried with citations, and promoted only after human review.

## Key Points

- `raw/` is allowed to be messy and source-shaped.
- `wiki/` is compiled structure: summaries, concepts, backlinks, and unresolved contradictions.
- `00-Inbox/AI-Output/` is a quarantine layer for agent answers and reports.
- Durable paths are reviewed team memory, not agent scratch space.

## Relationships

- **raw source capture** feeds **compiled wiki**
- **AI-Output drafts** require **human review**
- **reviewed conclusions** become **durable team memory**
